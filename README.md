# codex会话同步工具

一个用于浏览、修复、导入、导出和同步本地 Codex 会话记录的 Windows 桌面工具。

## 功能

- 浏览本地 Codex 会话目录
- 显示会话名称和会话 ID
- 修复单个会话文件或整个会话目录
- 导入会话并自动修复索引
- 导出全部会话或选中的会话
- 导入时检测重复会话 ID
- 重复 ID 可选择“替换”或“增量”
- “替换”：用导入会话覆盖本机同 ID 会话
- “增量”：保留原会话，并为导入会话生成新 ID，让两个内容相同但 ID 不同的会话同时存在
- 日志面板显示导入、导出、修复过程

## 适用场景

- Codex 会话文件存在，但会话列表中无法显示
- 手动复制会话文件后需要重新建立索引
- 需要备份、迁移或同步 Codex 历史会话
- 导入他人或其他设备的会话时，想避免覆盖已有同 ID 会话

## 技术说明

本项目使用 Python 开发，图形界面基于 Tkinter，图片资源处理使用 Pillow，Windows 可执行文件通过 PyInstaller 打包
